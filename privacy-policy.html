Privacy Policy

Explanium - Instant Text Explainer

Last Updated: July 2025

Quick Summary: Explanium respects your privacy. We don't collect, store, or share your personal information. Selected text is only sent to Google's Gemini API for explanations, and your API key stays on your device.

1. Information We Collect

1.1 What We DON'T Collect:

  • Personal information (name, email, phone, etc.)
  • Browsing history or website data
  • Location data
  • Device information beyond what's necessary for operation
  • Usage analytics or tracking data

1.2 What We DO Process:

  • Selected Text: Only the text you actively select for explanation
  • User Settings: Your extension preferences (stored locally)
  • API Key: Your Google Gemini API key (stored locally on your device)
  • Cache Data: Previously explained text and responses (stored locally)

2. How We Use Your Information

2.1 Selected Text Processing:

  • Selected text is sent to Google's Gemini API to generate explanations
  • Text is processed in real-time and not stored on our servers
  • Explanations are cached locally on your device for faster repeat access

2.2 Local Storage:

  • Extension settings are stored using Chrome's storage API
  • Cache data is stored locally to improve performance
  • Your API key is stored securely on your device only

3. Data Sharing and Third Parties

3.1 Google Gemini API:

  • Selected text is sent to Google's Gemini API for processing
  • Google's privacy policy applies to this data: https://policies.google.com/privacy
  • No personal information is included in API requests

3.2 No Other Third Parties:

  • We don't share data with advertisers, analytics companies, or other third parties
  • We don't use tracking pixels, cookies, or similar technologies
  • We don't sell or rent your information to anyone

4. Data Security

  • Local Storage: All personal data stays on your device
  • Encrypted Communication: API calls use HTTPS encryption
  • No Central Database: We don't maintain servers with user data
  • API Key Security: Your API key is stored using Chrome's secure storage

5. Your Rights and Controls

5.1 You Can:

  • Disable the extension at any time
  • Clear cached explanations through the settings
  • Remove your API key from the extension
  • Uninstall the extension to remove all local data

5.2 Data Retention:

  • Cached explanations are kept for 7 days then automatically deleted
  • Settings persist until you change them or uninstall
  • No data is retained after uninstallation

6. Children's Privacy

Explanium is suitable for all ages. We don't knowingly collect personal information from children under 13. If you're a parent and believe your child has provided personal information, please contact us.

7. Changes to Privacy Policy

We may update this privacy policy occasionally. Changes will be posted here with an updated "Last Updated" date. Continued use of the extension after changes constitutes acceptance of the updated policy.

8. Compliance

  • GDPR: We comply with EU data protection regulations
  • CCPA: We comply with California privacy laws
  • Chrome Web Store: We follow Google's privacy requirements

9. Technical Details

9.1 Permissions Used:

  • activeTab: Access current tab when you select text
  • storage: Store settings and cache locally
  • contextMenus: Add right-click menu option
  • host_permissions: Connect to Google Gemini API only

9.2 Data Flow:

  1. You select text on a webpage
  2. Text is sent to Google Gemini API via HTTPS
  3. Explanation is received and displayed
  4. Explanation is cached locally for faster repeat access
  5. No data is sent to any other servers
